Frutas, verduras, familiares, numeros y emociones. Algunas de estas paginas tendran ejercicios para practicar.

El primer apartado sera para aprender vocabulario, el segundo sera para practicar con una serie de preguntas y el tercer apartado sera para ver las respuestas una vez finalizada la segunda parte, para poder verificar nuestras respuestas si son correctas o no, Mucha Suerte!.
